Experiment 3 - Alternate worlds

Description

I am attempting at making a pixel world with the task of using autotiling, regenerative and grid code generation

Technical

I managed to create a chest cycle animation that switches variations of the '2' chest tiles in the tilemap, created walls that autotile based on the border between '.' tile and '_' default grass floor tiles and made hallways pointing out of bounds of the map. Placed chests to spawn on '_' and '1' furnaces/house tiles on the '.' grass tiles.

Reflection

I worked on this myself, I had a hard time intergrating my knowledge with working with sprite/tilemaps and applying myself into grid code generation.
